# Student Score Prediction

## Overview

Student Score Prediction is a Machine Learning project developed using Linear Regression to predict student exam scores based on study-related factors.

---

## Features

* Dataset preprocessing
* Exploratory Data Analysis
* Linear Regression Model
* Model Evaluation
* Flask Web Application
* User Input Prediction Interface
* Model Persistence using Joblib

---

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Flask
* Joblib
